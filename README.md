# Tela de Login com Efeito de Foco Dinâmico
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Uma interface de login moderna e dividida em dois painéis, desenvolvida com foco em interatividade e feedback visual utilizando HTML5, CSS3 e JavaScript puro (Vanilla JS).

## Sobre o Projeto
Este projeto apresenta um layout de duas colunas:
1. **Painel de Boas-vindas:** Um lado informativo que reage às interações do usuário.
2. **Formulário de Acesso:** Campos de entrada estilizados com validação nativa.

O diferencial deste projeto é a manipulação do DOM via JavaScript para criar um efeito de transição visual suave, destacando o formulário durante o preenchimento.

## Funcionalidades
* **Layout Split-Screen:** Divisão visual clara entre conteúdo institucional e funcional.
* **Efeito Blur Dinâmico:** O painel esquerdo sofre um desfoque suave quando o usuário foca nos campos de entrada, aumentando o foco no formulário.
* **Design Responsivo:** Preparado para se adaptar a diferentes tamanhos de tela.
* **Interatividade com JavaScript:** Uso de eventos de `focus` e `blur` para manipulação de estilos em tempo real.

## Tecnologias Utilizadas
* **HTML5:** Estrutura semântica com containers e formulários.
* **CSS3:** Estilização (referenciada via `css/style.css`).
* **JavaScript (Vanilla):** Lógica para manipulação de filtros CSS e transições de estado.

## Como visualizar
1. Clone este repositório ou baixe os arquivos.
2. Certifique-se de que a pasta `css` contém o arquivo `style.css`.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
